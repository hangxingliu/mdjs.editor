## 使用方法

1. 直接访问: <http://hangxingliu.github.io/mdjs.editor>  
2. 克隆当前仓库并检出`gh-pages`分支
	- `git clone https://github.com/hangxingliu/mdjs.editor.git`
	- `git checkout gh-pages`
3. 克隆当前仓库并构建源码(请先安装`node.js`与`npm`开发环境)
	- `git clone https://github.com/hangxingliu/mdjs.editor.git`
	- `npm i`
	- 构建生成的代码在`dist`目录下
