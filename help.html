快捷键

  • Ctrl+ N 新建一个Markdown文件
  • Ctrl+ O 打开一个本地的Markdown文件
  • Ctrl+ S 保存当前Markdown文件到本地
  • Ctrl+ / 弹出此帮助关于文档

输入样式快捷键


  • Tab 选定行行首加入Tab字符,或者输入Tab字符
  • Shift + Tab 选定行行首去掉一个Tab字符
  • Ctrl + B 选定内容加粗或去除加粗
  • Ctrl + I 选定内容斜体或去除斜体
  • Ctrl + H选定行变为标题行或者增大标题号
